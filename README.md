# 关于本项目

本项目的功能需求来源于影视站点后端开发，所以部分代码具有特定性。

## 豆瓣API（PHP部分）

本项目部分基于豆瓣电影官网，部分基于豆瓣未公开的API，没有用到APIKEY，但由于豆瓣官网的限制，单IP请求次数过多可能面临短时间封禁IP的惩罚。

## 豆瓣API

* 名人信息
* 影片短评
* 影片信息
* 影片影评
* 影片搜索
* 影片Tags分类
* Top250
* 豆瓣播放地址及资源网播放地址

## 爬虫（Python部分暂未写）

基于 MSSQL 数据库和 Python 遍历年份写入数据。
